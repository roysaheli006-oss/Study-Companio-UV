# **App Name**: Study Companion

## Core Features:

- Stress-Level Based News & Motivation Feed: Tailored content for each stress level: motivational messages, mental health tips, encouraging study reminders, positive news stories.
- Personalized Study Recommendations by Stress Level: Study recommendations based on user's stress level.
- Daily Streak System: Track consecutive study days; reward streaks with badges and motivational messages.
- Focus Lock Mode (Study Timer Protection): Prevent opening other apps while timer is active; block notifications except emergency calls.
- User Authentication & Profiles: Firebase Auth (Email + OAuth); save user preferences: study style, subjects, stress habits.
- Study Session Tracker: Timer with start/pause/stop, tags (subject, topic, mood/stress level); store session data in Firestore.
- Dashboard & Insights: Visual charts for study time, focus trends, weekly streaks; stress-aware insights, e.g., 'You focus best in low-stress mornings'.

## Style Guidelines:

- Overall Style: Calm, friendly, motivating, and modern
- Stress-Level Colors: Low Stress → Mint/Teal (#4CAF50) → energetic & focused; Moderate Stress → Amber/Yellow (#FFC107) → warm, balanced; High Stress → Coral/Red (#F44336) → calming, alerting gently
- Background: Soft off-white or light gray (#F9F9F9) → reduces visual fatigue
- Highlights / Buttons: Vibrant accent colors corresponding to stress level
- Typography: Clean, rounded sans-serif (e.g., Poppins, Inter) → readable & friendly
- Mood/Focus Icons: Emoji-style or minimal vector icons, consistent with stress colors
- Clean layout to improve readability and focus
- Subtle animations for transitions and feedback to maintain a calm and focused atmosphere
